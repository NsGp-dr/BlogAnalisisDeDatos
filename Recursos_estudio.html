Semana 1: Fundamentos de Análisis de Datos y Estadística

  1. Conceptos básicos de probabilidad y estadística:

    • Tutorial de Khan Academy sobre probabilidad y estadística: Khan Academy - Probabilidad y estadística
    • Libro recomendado: "Estadística para Dummies" de Deborah Rumsey.

  2. Muestreo y estimación:

    • Tutorial de Stat Trek sobre muestreo y estimación: Stat Trek - Muestreo y estimación
    • Libro recomendado: "Sampling Techniques" de Cochran, W.G. y otros.

  3. Correlación y regresión:

    • Tutorial de Statistic How To sobre correlación y regresión: Statistic How To - Correlación y regresión
    • Libro recomendado: "Applied Regression Analysis and Generalized Linear Models" de Fox, J.

  4. Ejercicios de probabilidad y estadística en Python y R:

    • Proyectos y desafíos de datos en Kaggle: Kaggle Projects
    • Curso en línea "Statistics for Data Science Using Python" en Udemy: Udemy - Statistics for Data Science Using Python

Semana 2: Manipulación de Datos y Análisis Exploratorio

  1. Manipulación de datos en Python y R:

    • Documentación oficial de pandas: pandas - Documentación
    • Documentación oficial de dplyr: dplyr - Documentación

  2. Limpieza y transformación de datos:

    • Tutorial de DataCamp sobre limpieza de datos con Python: DataCamp - Cleaning Data in Python
    • Tutorial de DataCamp sobre limpieza de datos con R: DataCamp - Cleaning Data in R

  3. Análisis exploratorio de datos:

    • Libro recomendado: "Exploratory Data Analysis" de Tukey, J.W.
    • Galería de visualizaciones de seaborn: seaborn - Gallery

  4. Conjuntos de datos reales para práctica:

    • Repositorio de conjuntos de datos de UCI Machine Learning Repository: UCI Machine Learning Repository

Semana 3: Análisis Estadístico Avanzado

  1. Pruebas de hipótesis, ANOVA y pruebas no paramétricas:

    • Tutorial de Laerd Statistics sobre pruebas estadísticas: Laerd Statistics - Statistical Tutorials
    • Libro recomendado: "Statistical Inference" de Casella, G. y Berger, R.L.

  2. Análisis multivariable y machine learning:

    • Tutorial de scikit-learn sobre machine learning con Python: scikit-learn - User Guide
    • Tutorial de DataCamp sobre machine learning con R: DataCamp - Machine Learning with R

  3. Interpretación y comunicación de resultados:

    • Artículo de Harvard Business Review sobre comunicación efectiva de análisis de datos: HBR - Data Storytelling

Semana 4: Procesamiento de Datos a Gran Escala y Visualización Avanzada

  1. Procesamiento de datos a gran escala:

    • Tutorial de Apache Spark: Apache Spark - Documentation
    • Tutorial de Hadoop: Hadoop - Documentation

  2. Visualización de datos avanzada:

    • Galería de ejemplos de visualizaciones con matplotlib: matplotlib - Gallery
    • Galería de ejemplos de visualizaciones con ggplot2: ggplot2 - Gallery

  3. Análisis de datos no estructurados:

    • Tutorial de NLTK para procesamiento de texto con Python: NLTK - Book
    • Tutorial de spaCy para procesamiento de texto con Python: spaCy - Documentation
